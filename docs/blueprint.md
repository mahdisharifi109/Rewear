# **App Name**: Rewear

## Core Features:

- Product Display: Display a product listing with image, name, price, and condition.
- Basic Product Search: Implement basic search functionality to find products by name or category.
- Simulated Shopping Cart: Add a shopping cart to allow users to 'add' desired products
- Informational Pages: Create static pages for 'About Us', 'Contact', and 'FAQ'.
- AI-Powered Product Suggestion: Suggest appropriate titles, descriptions and categories when the user uploads a new item, using a tool that analyzes image content to generate relevant details.

## Style Guidelines:

- Primary color: Soft Lavender (#D1B0FF) to evoke a sense of calm and trust, suitable for a marketplace.
- Background color: Light off-white (#F5F3FF), a desaturated tint of lavender, providing a clean and airy backdrop.
- Accent color: Dusty Rose (#E0A4FF), an analogous color that adds a touch of sophistication and draws attention to key interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and readable interface. The font should be used throughout for headlines and body text
- Use minimalist line icons to represent different product categories and actions.
- Grid-based layout for product listings to maintain a structured and visually appealing interface.
